# A-3 (Leetcode - 768)

## Solution
```bash
class Solution {
public:
    int maxChunksToSorted(vector<int>& arr) {
        long long sum1 = 0;
        long long sum2 = 0;
        int count = 0;
        int n = arr.size();
        vector<int> temp = arr;
        sort(temp.begin(), temp.end());

        for(int i=0;i<n;i++){
            sum1 = sum1 + temp[i];
            sum2 = sum2 + arr[i];
            if(sum1 == sum2) count++;
        }
        return count;
    }
};
```

keeping sum1 and sum2 as int type gives runtime error because of integer over, that's why long long is used.
